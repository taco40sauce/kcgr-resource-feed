# kcgr-resource-feed
Automated pipeline turning ham radio APRS ground reports (KCGR: Kershaw County Ground Report) into a real-time public emergency resource map — shelters, roads, water, fuel, power, and more. Built for KC4RC/Kershaw County EmComm on Raspberry Pi + Graywolf APRS software. Open source for any ARES/EmComm group to adapt.
